🗺️ Roadmap de Desarrollo

✔️

Versión Beta Inicial

Se implementa el sistema base que convierte el servidor en una fortaleza con generación pasiva de recursos.

✔️

Generación de Recursos

Los canales de voz permiten recolectar piedra, madera, agua y raciones según habilidades de D&D 5e.

✔️

Rol "Descansado"

Implementado el sistema de bonificación temporal al visitar la fogata, duplicando la recolección por 30 minutos.

✔️

Conexión con Base de Datos

MySQL integrado para registrar usuarios, recursos y estado de estructuras. Preparado para escalabilidad.

✔️

Renombrado de Canales en Tiempo Real

Los nombres de canales se actualizan automáticamente con el valor de recursos generados.

🛠️

Evitar duplicación de canales

Prevención automática de canales repetidos al usar comandos como /iniciar.

🛠️

Visualización de Fortaleza

Diseño de un sistema visual en Discord para mostrar estructuras construidas y progreso colectivo.

🎯

Módulo de Recompensas y Economía

Implementación de un sistema económico donde los recursos puedan canjearse por mejoras, roles o ítems.

🎯

Expansiones Narrativas

Desbloqueo de nuevas zonas (como minas, talleres, bibliotecas) con efectos de juego y ambientación.

🚀

Publicación Oficial y Documentación

Lanzamiento del bot con documentación para su uso, comandos, y guía de administración para comunidades.
